  3. Java中数值类型之间的合法转换

    上面是Java基本类型之间的合法转换，boolean类型与int类型之间无法转换，实心箭头代表无信息丢失，注意int转换成double类型无信息丢失。
    当一个二元运算符连接两个值的时候，JAVA的计算规则是：
    • 如果两个操作数中有一个是double类型的话，另一个操作数转换成double类型。
    • 否则，如果其中一个是float类型的话，另一个操作数转换成float类型。
    • 否则，如果其中一个是long类型的话，另一个操作数转换成long类型。
    • 否则，两个操作数都将转换成int。
  4. 当位运算符作用到boolean类型的时候，不采用短路运算符，即两边的表达式都会进行求值操作。
    >>>代表用0填充高位，>>代表用符号位填充高位，不存在<<<运算符。

  6. Scanner的常用API，Java可以通过printf格式化输出，也可以通过String.format创建格式化字符串。
