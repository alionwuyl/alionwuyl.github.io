JAVA基础之File类

发表于 2020-04-12

这篇文章记录File类的相关用法。

定义

JDK 13上面说是：An abstract representation of file and directory pathnames. 即文件和目录路径名的抽象表示。
主要用于文件、目录的创建、查找和删除等操作。

File类中的静态常量

  • pathSeparator：系统相关的路径分隔符，为方便起见，以字符串形式表示。
  • pathSeparatorChar：系统相关的路径分隔符。
  • separator：系统相关的默认名称分隔符，为方便起见，以字符串形式表示。
  • separatorChar：取决于系统的默认名称分隔符。
    一个是单个路径中文件层次目录的分隔符，比如windows中是\，Linux中是/，另一个是多个路径之间的分割符，windows是;，linux是:。然后又分为字符和字符串表示。

构造方法

  1. 可以直接传入绝对路径名创建一个File对象，也可以使用相对路径，相对路径默认为当前用户目录来分析相对路径名，此目录由系统属性user.dir指定，通常是Java虚拟机的调用目录，在不同的地方启动会是不同的user.dir，最好使用绝对路径。
  2. 可以使用父级和自己共同创建。
  3. 使用URL进行创建。

常用方法

  1. file.getName();
  2. file.getPath();
  3. file.createNewFile();
  4. file.mkdir();
  5. file.length();
  6. file.list(); //返回字符串数组
  7. file.listFiles(); //返回File对象数组
  8. file.delete();
  9. file.exists();
  10. file.isDirectoty();
  11. file.isFile();
